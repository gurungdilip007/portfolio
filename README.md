## What to Customize

1. **Personal Information**:
   - Update your name, email, and phone number in the HTML
   - Change "Your City, Country" to your actual location

2. **Skills**:
   - Modify the skill tags to match your expertise
   - Add or remove skill categories as needed

3. **Projects**:
   - Replace the example projects with your real projects
   - Add links to your GitHub repositories
   - Update project descriptions

4. **Social Links**:
   - Add your GitHub profile URL
   - Add your LinkedIn profile URL
   - Add your Twitter/X profile URL
   - Add your CodePen or portfolio links

## Customization Tips

### Adding More Skills
Go to the Skills section in `index.html` and add new skill tags inside any category:
```html
<span class="skill-tag">New Skill</span>
```

### Adding More Projects
Duplicate a project card and update:
- Icon (change the `fas fa-*` class)
- Project title
- Description
- Technology tags
- Project link

### Changing Colors
Update the CSS variables in `style.css`:
```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    /* ... other colors ... */
}
```
